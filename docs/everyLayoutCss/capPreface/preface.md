# PREFACIO

Esta es la versión ebook de Every Layout y contiene todo el mismo contenido. La hicimos para ti porque pensamos que te podría gustar leer sobre diseño CSS en este formato y sin conexión.

Un libro EPUB no puede hacer o mostrar todo lo que un sitio web puede, y los demos interactivos se reemplazan con enlaces de vuelta al sitio web. Para ver todas estas páginas y demos, tienes que comprar la versión completa de Every Layout. Si estás viendo este libro, esperamos que ya lo hayas hecho. Después de la compra, un enlace al sitio completo y desbloqueado acompañará al enlace de este libro en un correo electrónico.


Todos los enlaces externos en este libro están marcados con un símbolo/carácter ↗. Si ves un enlace sin el sufijo ↗, apunta a una sección dentro del propio libro. El libro ha sido probado para renderizar y comportarse como se espera en las últimas versiones de iBooks de Apple y de Calibre ebook manager ↗.

## ¡Segunda edición!

De hecho, esta es la segunda edición de Every Layout, completada alrededor de junio de 2021. Esta edición convierte varios layouts para usar la propiedad gap, la cual ha llegado a ser ampliamente soportada tanto con Flexbox como con Grid. Usar gap simplifica muchos layouts y los hace más fáciles de entender.
